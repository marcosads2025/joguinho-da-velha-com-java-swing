Sobre o projeto

Este é um jogo clássico de Pedra, Papel e Tesoura, onde o usuário joga contra o computador.

A aplicação possui uma interface gráfica simples e intuitiva, permitindo que o usuário escolha uma das opções e visualize o resultado da rodada em tempo real.

🖥️ Interface
Tela com botões:
Pedra 🪨
Papel 📄
Tesoura ✂️
Exibição do resultado:
Vitória
Derrota
Empate
⚙️ Tecnologias utilizadas
Java
Swing (Javax Swing)
Programação Orientada a Objetos (POO)
🧠 Conceitos aplicados
Estruturas condicionais (if/else)
Geração de números aleatórios (Random)
Manipulação de eventos (ActionListener)
Interface gráfica com Swing
Separação de lógica e interface
🚀 Como executar o projeto
Clone o repositório:
git clone https://github.com/seu-usuario/seu-repositorio.git
Abra o projeto em sua IDE (IntelliJ, Eclipse, etc.)
Execute a classe principal:
Main.java
🎯 Regras do jogo
Pedra vence Tesoura
Tesoura vence Papel
Papel vence Pedra
